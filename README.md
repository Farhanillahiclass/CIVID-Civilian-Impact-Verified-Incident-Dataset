# CIVID

CIVID is a verified conflict-impact dataset project for structured research on civilian impact events.

## Structure

- data/phase1_palestine/: starter Phase 1 CSV files
- data/phase2_sudan/: placeholder for future phase data
- schema/: JSON schema for CIVID records
- src/: project code and utilities
- notebooks/: exploratory notebooks

## Setup

Run the bootstrap script from the project root:

```bash
bash setup.sh
```
