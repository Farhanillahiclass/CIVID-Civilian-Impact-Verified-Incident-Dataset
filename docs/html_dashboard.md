# CIVID — HTML Dashboard

Generated by `scripts/generate_html_dashboard.py` → `exports/civid_dashboard.html`
(self-contained; Chart.js + Leaflet from CDN; works offline except map tiles/charts libs).

## Features

- **Filters:** phase, country, verification status, victim role, age group (child/adult),
  from-date. Charts re-render on filter change.
- **Summary cards:** events, verified, needs-review, total/children/women/doctors/journalists/
  commanders killed, arrests, detentions, persons.
- **Charts:** events timeline (by month), verification status, events by phase, victim-role
  breakdown, aggregate intelligence (news metrics).
- **Map:** Leaflet map (plots only where lat/lon exist; location text is stored otherwise).
- **Famous personalities section:** image gallery + bios, sourced and ethics-gated.
- **News section:** authentic source-linked stories from `news_intelligence.csv`.
- **Responsive:** single-column layout on mobile (≤820px).

## Ethics

- No victim/casualty imagery. No identifiable images of minors unless public, relevant, safe.
- Every number traces to `exports/` → `sources.csv`. Disclaimer footer on every page.
- Aggregate vs single-incident figures are labelled; `derived_is_aggregate` is honoured.

## Regenerate

```bash
python scripts/build_exports.py
python scripts/generate_news_intelligence.py
python scripts/generate_html_dashboard.py
```
